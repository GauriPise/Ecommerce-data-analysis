# 🛒 Ecommerce Data Analysis

## 📌 Problem Statement
Analyze e-commerce sales data to identify customer purchasing behavior, product performance, and revenue trends to support business decisions.

## 📊 Dataset Info
- Orders data (Order ID, Product, Category, Sales, Quantity)
- Customer data (Customer ID, Region)
- Order Date (Time-based analysis)

## ⚙️ Steps Performed
### Python (Pandas)
- Data cleaning (handled null values, removed duplicates)
- Converted date columns into Month & Year
- Aggregated sales by category and region

### Visualization
- Sales trend over time
- Category-wise performance analysis

## 📈 Key Insights ⭐
- Few categories contribute majority of revenue
- Sales show strong seasonal trends
- Certain regions dominate overall sales
